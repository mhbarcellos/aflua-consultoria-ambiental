# Aflua Consultoria Ambiental

Landing page institucional da **Aflua Consultoria Ambiental**, desenvolvida para apresentar a empresa, seus serviços, diferenciais e canais de contato.

Site publicado em:

https://aflua.netlify.app/

## Sobre o projeto

A página foi criada em HTML, CSS e JavaScript puro, sem dependências externas.  
O objetivo é oferecer uma presença digital leve, responsiva e otimizada para empresas que buscam soluções em consultoria ambiental.

## Principais seções

- Hero institucional
- Quem somos
- Nossos serviços
- Como atuamos
- Por que escolher a Aflua
- Contato com formulário e WhatsApp
- Rodapé com informações institucionais

## Serviços apresentados

- Licenciamento e regularização ambiental
- Gestão de resíduos
- Monitoramentos e relatórios técnicos
- Auditorias e diagnósticos ambientais
- Educação ambiental e treinamentos
- Sustentabilidade e ESG

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Netlify
- Google Search Console

## Recursos implementados

- Layout responsivo para desktop, tablet e celular
- Animações de rolagem com JavaScript
- Parallax sutil em elementos decorativos
- Header com efeito ao rolar
- Formulário com validação front-end
- Link direto para WhatsApp
- Link para Instagram
- SEO básico com meta tags
- Open Graph e Twitter Card
- Sitemap XML
- Robots.txt
- Verificação no Google Search Console

## Como rodar localmente

Clone o repositório e acesse a pasta do projeto:

```powershell
git clone https://github.com/mhbarcellos/aflua-consultoria-ambiental.git
cd aflua-consultoria-ambiental

Abra o arquivo index.html diretamente no navegador ou rode um servidor local:

python -m http.server 5173

Depois acesse:

http://127.0.0.1:5173/
Deploy

O deploy é feito automaticamente pelo Netlify a partir da branch main do GitHub.

Sempre que houver um novo push para a branch main, o Netlify publica uma nova versão do site.
