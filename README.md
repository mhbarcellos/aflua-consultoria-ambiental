# Aflua - Consultoria Ambiental

Landing page responsiva desenvolvida para a Aflua Consultoria Ambiental.

## Sobre o projeto

Este projeto apresenta uma página institucional para uma consultoria ambiental, com foco em comunicação clara, design responsivo e apresentação dos serviços oferecidos.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como visualizar

Abra o arquivo `index.html` no navegador.
